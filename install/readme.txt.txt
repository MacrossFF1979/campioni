Installazione sito e-commerce.

1. Installare xampp
2. Creare la directory \e-commerce\ in htdocs
3. Copiare tutti i file in C:\xampp\htdocs\e-commerce\
4. Lanciare la pagina di installazione digitando l'indirizzo sulla barra del browser.
    Potete copiare la seguente riga: http://localhost/e-commerce/install/creazioneDB.php

- La pagina creerà il database NEGOZIO e i due database USER E OGGETTI.
- La pagina creerà inoltre l'utente Admin, che non può essere cancellato.
